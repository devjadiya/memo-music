# **App Name**: MemoVerse

## Core Features:

- Homepage Showcase: Display artist profiles, music videos, and recent news to capture user attention immediately.
- Studio Services: Detail the studio's features, rental options, and equipment available.
- Team Directory: Display profiles of key team members with their expertise listed.
- PR and Fame Development: Describe how MEMO MUSIC builds visibility through PR, media, and branding.
- Independent Artist Services: Offer accessible options for one-time or short-term project assistance.
- AI Music Analyzer: Tool to help artists receive generative-AI driven insights into trends. Provides data driven ideas to improve artistic and commercial performance of a music track before formal release
- Contact Form: Streamlined submission to forward all artist inquiries.

## Style Guidelines:

- Primary color: Deep blue (#1A237E), echoing sophistication and mirroring the brand logo.
- Background color: Black (#000000), creates a sleek and luxurious backdrop.
- Accent color: Electric blue (#7DF9FF), adds vibrant contrast and highlights key elements.
- Headline font: 'Playfair', a modern sans-serif similar to Didot, for elegant and high-end headings. Body text: 'PT Sans', a humanist sans-serif.
- Use minimalist icons that emphasize artistry, fame, and global reach.
- Employ a grid-based layout for structured and stylish presentation.
- Subtle, sophisticated transitions that exude quality.
