# MEMO MUSIC Website

This project is a modern, responsive, and visually rich corporate website for MEMO MUSIC, a premier music production company. The site is designed to be a sleek, single page experience that showcases the company's brand, services, and portfolio.

## Features

*   Fully responsive design optimized for mobile, tablet, and desktop devices.
*   A dynamic hero section featuring a background video to immediately capture user attention.
*   Multiple distinct sections detailing the company's services, team, studio, and mission.
*   An interactive mobile navigation menu for a seamless user experience on smaller screens.
*   A functional contact form for user inquiries.
*   An embedded Google Map to display the company's location.

## Technology Stack

The project is built with a modern and robust technology stack:

*   **Framework:** Next.js
*   **Language:** TypeScript
*   **Styling:** Tailwind CSS
*   **UI Components:** ShadCN UI
*   **Icons:** Lucide React
*   **Form Handling:** FormSubmit
